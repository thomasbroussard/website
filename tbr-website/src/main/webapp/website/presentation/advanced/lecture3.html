Servlets

Servlets are like JSP... in fact JSP are Servlets where HTML output is favored.

First advice
  • If you want to do basic presentation, choose a JSP
  • If you want to perform treatments, call Servlet
  • The link between each of them can be made through requests/responses or session

Servlets (2)

Differences between 2.5 and 3.0 servlets implementation

  • In the former implementation (2.5) what one calls "Mapping" is done through the web.xml file
  • In the new implementation, this mapping is done directly in the file thanks to the @WebServlet annotation
