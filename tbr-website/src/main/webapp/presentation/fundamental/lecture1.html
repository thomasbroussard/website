Object Oriented Programming : why?

What already exists: Procedural programming

  • A lot of code to make pieces of program work together
  • A technical approach that lessen the code comprehension
  • Big programs are difficult to maintain

Object Oriented Programming : (human) definition

The definition of the Object Oriented Programming, is that this kind of programming uses object

This said, what is an Object?

  • Objects are a conceptual representation of existing things
  • This makes the programming easier to approach because things are clearer for the developper
